This folder contains all components required to build the terrain rendering sample.

Visual studio solutions (TerrainRender_rel1_2008.sln and TerrainRender_rel1_2010.sln) are located in Terrain_rel1 folder.

Prerequisites:
* Microsoft DirectX SDK June2010 is installed
* Visual studio paths are properly configured:
** "$(DXSDK_DIR)Include" is added to Include Directories
** "$(DXSDK_DIR)Lib\x64" or "$(DXSDK_DIR)Lib\x86" is added to Library Directories
** "$(DXSDK_DIR)Utilities\bin\x64" or " $(DXSDK_DIR)Utilities\bin\x86 " is added to Executable Directories
